### 降权项说明

以下类型函数虽存在同名或相似调用链，但**不构成强同源证据**：

1. **POSIX 标准接口包装函数**（`sys_getpid`、`sys_gettid`、`sys_exit`、`sys_yield` 等）：函数名由 Linux ABI 强制规定，实现体为 1~3 行的标准调用模式，AST 可变性为零。
2. **RISC-V 特权指令封装函数**（`set_kernel_trap_entry`、`set_user_trap_entry`、`enable_timer_interrupt`）：仅包含单条 CSR 写入指令，由硬件规范决定。
3. **TCB 字段访问器**（`getpid`、`gettid`）：返回结构体字段的单行函数，AST 由数据结构定义决定。
4. **常量存根函数**（`sys_getuid`、`sys_geteuid`、`sys_getgid`、`sys_getegid`）：教学 OS 中通常返回常量 0，无创造性表达。
5. **跨语言对比项**（目标 Rust 用户库 vs 参考 C 用户库）：语言、ABI、调用约定均不同，仅因都封装了 `syscall` 指令而被列为证据，属于跨板块硬凑。
6. **rCore 框架公共代码**（`new_kernel`、`fork` 等）：源自相同的教学框架，属于公共依赖而非代码复用。
