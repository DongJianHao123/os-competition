## 第四部分：分模块架构说明

### MM（内存管理）
- **已体现**：物理页帧分配器（`FRAME_ALLOCATOR`）、内核堆分配器（带 trace/perf 统计）、用户地址空间从 ELF 创建、页表操作、缺页异常处理、trap context ppn 分配。
- **证据锚点**：`os/src/mm/frame_allocator.rs#frame_alloc_uninit`、`os/src/mm/heap_allocator.rs#dealloc`、`os/src/mm/address_space.rs#alloc_user_res_with_trap_ppn`、`os/src/hal/arch/riscv/trap/mod.rs#trap_handler`。
- **未体现**：swap、页替换算法、NUMA 内存管理、大页（huge page）等底层证据未体现。

### PROC（进程管理/任务调度）
- **已体现**：进程与线程控制块、`clone`/`execve`/`waitid`/`exit`、futex、信号投递与返回（`do_signal`、`sys_sigreturn`、sigmask）、POSIX timer/itimer、System V IPC（sem/msg/shm）、POSIX mq、命名空间（mount/ipc/net，通过 `sys_unshare`/`sys_setns`）、rlimit/prctl/capabilities/seccomp、pidfd、`process_vm_transfer`、`kcmp`、procfs pid/task/fd 节点。
- **证据锚点**：`os/src/task/task.rs#sys_clone`、`os/src/task/processor.rs#run_tasks`、`os/src/syscall/process/exec.rs#sys_execve`、`os/src/syscall/process/ipc.rs#sys_semctl`、`os/src/syscall/process/clone.rs#sys_unshare`、`os/src/task/signal/mod.rs#do_signal`。
- **任务调度**：传统就绪队列 + `WaitQueue` + `block_current_and_run_next` 式调度证据充分；**异步运行时（async/await executor、waker、ring_fifo 等）底层证据未体现**。
- **未体现**：cgroup 资源控制、ptrace、审计、完整的实时调度器等底层证据未体现。

### FS（文件系统）
- **已体现**：VFS 抽象层（mount/file/inode）、ext4（目录查找、rename、unlink、symlink、read_at）、tmpfs、procfs（pid/task/fd/net 等动态节点）、devfs（tty、pipe）、initramfs（newc 格式解压）、eventpoll、fcntl、splice、fallocate、mount/umount、syncfs。
- **证据锚点**：`os/src/fs/vfs/mount.rs`、`os/src/fs/vfs/file.rs`、`os/src/fs/ext4/ext4fs.rs#find`、`os/src/fs/tmpfs/mod.rs#rename`、`os/src/fs/initramfs.rs#unpack_newc`、`os/src/fs/eventpoll.rs#sys_epoll_pwait`。
- **未体现**：日志恢复、quota、NFS/FAT/exFAT/Btrfs、LVM、文件系统加密等底层证据未体现。

### DEV（设备驱动）
- **已体现**：virtio-blk-pci（DMA 缓冲区管理）、veth（虚拟网卡对）、tty 控制台。
- **证据锚点**：`os/src/drivers/block/virtio_blk_pci.rs#unshare`、`os/src/drivers/net/veth.rs#new`、`os/src/fs/dev/tty.rs#read_at`。
- **未体现**：USB、NVMe、AHCI、SDHCI、GPU、音频、通用 DMA 框架等底层证据未体现。

### SYSCALL（系统调用）
- **已体现**：集中式系统调用分发器 `syscall`（内部调用链规模 275），覆盖文件、进程、网络三大类；参数通过 `UserPtr`/`UserCString`/`UserBufferWriter` 等安全封装拷贝。
- **证据锚点**：`os/src/syscall/mod.rs#syscall`、`os/src/syscall/fs.rs#sys_openat`、`os/src/syscall/process/ipc.rs#sys_msgctl`、`os/src/net/syscall/bind.rs#sys_bind`。
- **未体现**：完整 eBPF 验证器、内核态 JIT 等底层证据未体现。

### TRAP/IRQ（中断异常）
- **已体现**：RISC-V 与 LoongArch64 的 trap handler、异常原因解码（`estat::cause`）、访存指令解码（`mem_access::get_addr`）、内核态 trap 处理。
- **证据锚点**：`os/src/hal/arch/loongarch64/trap/mod.rs#trap_handler`、`os/src/hal/arch/loongarch64/trap/mod.rs#trap_from_kernel`、`os/src/hal/arch/loongarch64/register/base/estat.rs#cause`、`os/src/hal/arch/loongarch64/trap/mem_access.rs#get_addr`。
- **未体现**：中断控制器（PLIC/APLIC/IOAPIC）驱动细节、IRQ 路由、MSI/MSI-X、时钟中断源等底层证据未体现。

### NET（网络协议栈）
- **已体现**：socket 层（inet stream/datagram/raw、unix、netlink、packet）、TCP/UDP/Raw 通过 smoltcp 承载、路由表、邻居/ARP、veth 对、`SIOCGIF` ioctl、netlink route（newlink/newaddr/dellink）、socket 上的 epoll 事件。
- **证据锚点**：`os/src/net/config.rs#poll_once`、`os/src/net/socket/inet/stream/mod.rs#connect`、`os/src/net/socket/netlink/route/link.rs#handle_newlink`、`os/src/net/routing.rs#route_output`、`os/src/net/neighbour.rs#try_capture_arp_reply`。
- **未体现**：原生 TCP/IP 协议实现（依赖 smoltcp）、WiFi、IPv6 高级特性、IPSec、TLS 等底层证据未体现。
