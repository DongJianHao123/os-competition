### 第三方库与工具说明
- **网络协议栈核心**基于 `smoltcp`（`arceos/modules/axnet/src/smoltcp_impl/` 为封装层）。
- **文件系统**中 FAT 等可能依赖第三方 crate；`compat/oscomp-runner` 使用第三方 ext4 crate 作为宿主机镜像生成工具，不属于内核主体。
- **C 兼容库** `arceos/ulib/axlibc/c` 包含大量 musl/newlib 风格的数学/字符串/stdio 实现，部分函数为未实现桩。
- `arceos/tools/bwbench_client` 为宿主机网络测速工具，与内核驱动无关。
