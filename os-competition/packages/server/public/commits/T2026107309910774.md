#### 3.1.1 详细模块介绍
项目围绕"Phytium KVM/virtio 边缘服务虚拟化"展开，包含三大方向：（1）系统虚拟化——通过 KVM 创建 VM，配置 virtio 串口、PCI 设备直通（SMMU/ITS）；（2）外设虚拟化——实现设备隔离、DMA 保护、故障注入与隔离验证；（3）容器虚拟化——在 VM 内运行 Podman 容器工作负载，实现资源隔离与性能基准测试。所有模块均提供完整的 Shell/Python 脚本和 Markdown 文档，并配有架构图、数据流图等可视化材料。
