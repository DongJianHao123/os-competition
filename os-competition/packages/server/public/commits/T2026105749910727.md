#### 3.1.1 详细模块介绍
项目 Agent eBPF Filter 是一个 Linux-first 的 AI Agent 观测与控制平面，由 stevessr 独立开发。项目采用六层架构（L0-L5），核心模块包括：

1. **eBPF 内核采集模块**：使用 CO-RE 兼容的 eBPF 程序，通过 tracepoint 捕获 50+ 系统调用事件（execve、openat、connect、sendto、recvfrom 等），以及 sched_process_fork/exec/exit 进程生命周期事件和 tcp_connect/tcp_close 网络流事件。采用 ring buffer 传递事件，per-CPU 统计，支持 PID/comm/path/prefix 多级过滤。

2. **Go 后端模块**：基于 Gin 框架，包含完整的应用上下文管理、特征门控系统、路由注册、WebSocket 实时推送、后台任务（内核事件解码、广播、风险反馈、UDS 服务器、GC 循环、GeoIP 等）。438 个单元测试全部通过，50+ API 端点。

3. **Vue 前端模块**：Vue 3 + Vite + TypeScript，包含 Dashboard、Network、Execution Graph、AgentSight、TLS Capture、Config 等多个视图。支持实时事件流、网络流分析、进程树观察、录制回放等功能。存在 5 个 TypeScript 编译错误待修复。

4. **Wrapper 命令拦截模块**：Go 实现的 CLI 包装器，通过 UDS 与后端通信，支持命令策略执行、参数传递（--user/--cwd/--observer），实现 Agent 启动的自动观测。

5. **cgroup 网络阻断模块**：eBPF cgroup 程序，支持 TCP/UDP connect/sendmsg 阻断，23/24 测试通过，1 个 UDP connected-socket 边缘用例失败。

6. **BPF LSM 执行阻断模块**：BPF LSM 程序，实现 15 个 hooks 的文件/执行阻断，smoke test 通过。

7. **TLS 捕获模块**：通过 uprobe 捕获 SSL_read/SSL_write，支持 BoringSSL 精确匹配和 OpenSSL 通配符匹配。默认关闭，需用户主动启用。已修复 1970 时间戳问题（bpf_ktime_get_ns() 单调时间转换）。

8. **Agent 语义关联模块**：通过 PID 注册、native hooks、Python/JS adapters 实现 Agent 行为的语义关联，支持 Claude Code、Gemini CLI、Codex、GitHub Copilot、Kiro CLI、Augment、Antigravity CLI 等。
