#### 3.1.1 详细模块介绍
本项目在StarryOS（Rust操作系统）上实现了一套完整的智能捡球机器人系统，以RK3588 SoC为主控平台。系统架构包括用户态应用（tennis）、StarryOS内核（USB驱动、NPU驱动、tty_serial）和硬件层（USB摄像头、RK3588 NPU、ESP32底盘）。

**UVC摄像头采集模块**：基于libusb和libuvc库，通过UVC协议从USB摄像头采集MJPEG视频流，支持640×480@30fps分辨率。实现了帧捕获、MJPEG解码等功能，代码位于app/aka-rk3588/capture/目录。

**NPU推理模块**：使用YOLOv8n架构，训练集11268张，验证集684张，测试集285张，目标类别为tennis_ball。模型转换为RKNN格式时通过Split节点解决量化精度问题。推理管线包括MJPEG解码、Letterbox缩放、RKNN NPU推理（~37ms/帧）、NMS后处理。代码位于app/aka-rk3588/detect/和app/tennis-train/目录。

**用户态应用模块**：实现状态机（CHASE_BALL→GRAB→FIND_BUCKET→APPROACH_BUCKET→DEPOSIT），运动控制采用差速转向和动态调速算法。代码位于app/aka-rk3588/tennis.cpp（约31KB）。

**电机控制模块**：通过UART与ESP32通信，控制底盘电机和舵机。实现PWM和UART双模式电机驱动，ESP32固件完成PID控制和编码器反馈。代码位于app/aka-rk3588/motor/和hardware/esp32_base_control/目录。

**操作系统适配**：包括tty_serial驱动、NPU驱动修复和BusyBox rootfs构建。StarryOS端到端延迟155.73ms（约6fps），瓶颈主要在Letterbox（69.65ms，RGA未打通）和NPU outputs_get（54.99ms，cache sync低效）。
