#### 3.1.1 详细模块介绍
项目基于开源X-OmniClaw框架二次开发，并移植了mobile-use的视觉多角色能力。核心架构分为三层：Android宿主层（Kotlin，负责无障碍服务、截图、端侧感知YOLO/OCR/显著性融合、技能加载、深链录制）、桥接层（bridge契约）、决策大脑层（端侧Python，负责路由分流、无障碍循环、视觉多角色循环）。关键设计亮点包括：双路协同（无障碍路线快准便宜 + 纯视觉VLM路线广覆盖慢贵互补）、感知融合用IoU阈值0.15对齐YOLO框与无障碍节点、深链将多步操作压为一跳、完成度判定避免模型空转。
