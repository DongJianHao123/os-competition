#### 3.1.1 详细模块介绍
项目基于RISC-V 64架构，使用Rust语言开发，运行在QEMU模拟器上。内核基础模块基于rCore框架二次开发，实现了从启动到Shell交互的完整链路。教学服务层是项目的核心创新，包含16个操作系统标准章节、16个配套实验、160道题库，以及练习/考试/学习统计/薄弱点分析/知识点搜索等完整功能。所有教学模块在内核态Shell中实现，无需用户态支持。用户程序层保留了rCore框架原有的40+个测试程序，为后续扩展预留接口。
