#### 3.1.1 详细模块介绍
项目实现了一个多机器人任务规划与调度求解器，核心代码位于solver/generate_baseline.py（991行）。该求解器读取场景配置（地图、机器人、任务、状态），通过多种调度策略（束搜索、贪心调度、固定顺序调度）为不同tier（basic/deadlock/heterogeneous）生成动作序列，并使用外部checker评估方案质量，最终输出最优动作计划。主要功能包括：基于Dijkstra/SIPP的路径搜索、OccupancyTable时空占用管理、死锁避免与临界区处理、异构机器人协同处理、评分器适配与动作压缩。
