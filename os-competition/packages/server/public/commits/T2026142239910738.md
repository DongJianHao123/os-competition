#### 3.1.1 详细模块介绍
项目是一个用Rust编写的轻量级JavaScript引擎，采用经典的Lexer→Parser→Runtime→Interpreter架构。词法分析器（lexer.rs）支持完整的JS词法规则，包括模板字面量、正则表达式、Unicode转义等；语法分析器（parser.rs）采用递归下降+优先级爬升算法，支持ES2024级别的语法特性；运行时（runtime.rs）实现了基于原型链的对象模型、作用域环境栈、闭包支持；解释器（interpreter.rs）递归遍历AST执行，支持各类语句和表达式；内置对象（builtins.rs）实现了Array和Object的常用原型方法，部分为简化实现。整体代码量约22万字节，模块划分清晰，代码质量较高。
