#### 3.1.1 详细模块介绍
**系统移植模块**：基于昇腾官方适配仓库（ascend-open-harmony, 310b_rc_24.1.rc2_5.0.3），在Ubuntu 20.04 x86_64主机上搭建交叉编译环境，通过repo init拉取OpenHarmony 5.0.3源码，合入适配代码（替换device_board_ascend/device_soc_ascend/vendor_ascend），运行ascend_env.sh自动打补丁，全量编译后通过formatSD310B.sh制卡烧录。解决了交叉编译工具链路径配置、HDC补丁合入失败（dtsi权限不匹配）、ArkTS编译OOM（配置8G swap）三个关键问题。最终系统成功启动到图形桌面，HDMI 1080P显示正常，USB鼠标/键盘/触摸屏可用，HDC调试通道正常，完成全部25分评分项。

**CANN移植模块**：下载CANN 8.2.RC1 ARM64包，安装Runtime（最小化推理运行环境）和OPP算子包，部署Python推理依赖（numpy, torch, torch_npu, ais_bench, opencv-python等），配置环境变量脚本cann_env.sh。通过npu-smi info验证NPU状态为Health=OK（芯片310B1，驱动版本24.1.rc2），NNRT版本8.2.RC1，Python接口InferSession导入成功。文档记录了6个常见问题（库找不到、numpy版本不兼容、npu-smi不在PATH等）及解决方案。完成全部30分评分项。

**YOLOv5推理模块**：基于YOLOv5s模型，通过ONNX→OM格式转换（CANN格式），实现三种推理方式：1）CLI命令行推理（main.py，输入图片→letterbox预处理→CANN InferSession推理→NMS后处理→绘制检测框→保存结果，推理耗时66.7ms）；2）HTTP推理服务（serve.py，监听8080端口，提供GET /health和POST /detect接口）；3）OpenHarmony ArkUI图形APP（qizhi01_yolo_app，通过HTTP调用后端推理，支持从相册选择图片/加载样例图片，实时显示检测结果）。对world_cup.jpg测试图片检测到3个目标（person 0.89, person 0.72, bottle 0.65）。

**性能优化模块**：针对系统启动时间和内存占用进行优化。启动优化：降低内核日志等级（loglevel 5→3，减少日志I/O阻塞）、非关键init服务（uart/bluetooth/wifi/camera）设为ondemand延迟启动、CANN驱动并行加载优化，启动时间从52.35s降至45.65s（-12.8%）。内存优化：日志等级调整+服务延迟+开机drop_caches（清理pagecache/dentries/inodes），内存占用从1.7GB降至1.3GB（-23.5%）。内核裁剪探索（关闭调试跟踪、IPv6/无线/蓝牙、不用的文件系统和驱动、调整HZ参数）实测收益<0.5s，非主要优化手段。

### 4.2 遭遇的代码问题
| 问题描述 | 解决方法 |
| --- | --- |
| 交叉编译工具链aarch64-target-linux-gnu-gcc找不到 | 手动创建工具链软链接，export PATH添加工具链路径 |
| HDC补丁自动应用失败（dtsi文件权限644 vs 755） | 放弃自动打补丁，手动修改对应dtsi文件 |
| ArkTS编译OOM（内存溢出被OOM killer终止） | 配置8GB swap交换空间 |
| numpy 2.x与PyTorch 2.1.0不兼容（Could not infer dtype of numpy.float32） | 降级numpy至1.26.4 |
| libascendcl.so找不到（LD_LIBRARY_PATH未包含CANN库目录） | export LD_LIBRARY_PATH添加CANN库路径 |
| npu-smi命令找不到（位于/usr/local/sbin/不在PATH中） | export PATH添加/usr/local/sbin或使用完整路径 |
| Python包安装失败（x86_64 wheel不能在ARM64上安装） | 使用manylinux_2_17_aarch64或linux_aarch64版本wheel包 |
| NPU设备Health异常（驱动未加载/设备节点不存在/版本不匹配） | 检查lsmod、/dev/davinci_manager、npu-smi --version逐步排查 |
| env.sh变量展开问题 | 改用Python脚本写文件 |
