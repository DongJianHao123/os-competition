#### 3.1.1 详细模块介绍
项目围绕 ROS2 实时通信场景下的 DDS 调优展开。核心工作包括：1）设计并实现基线测试节点，通过节点内部加压（CPU/内存/延迟模拟）复现赛题场景；2）构建 Fast-DDS 2.13.6 overlay 并集成到 ROS2 的 rmw_fastrtps 中间件；3）针对不同话题定制 QoS 策略（如 /cmd_vel 使用 BEST_EFFORT + KEEP_LAST，/scan 使用 RELIABLE + KEEP_ALL）；4）设计 2×2 自动化测试矩阵，对比 baseline 与 fastdds 在有无 QoS 优化下的性能差异；5）收集并分析延迟、丢包率、CPU、RSS 等指标，验证优化效果。
