#### 3.1.1 详细模块介绍
**Raw Socket / ICMP 支持**：在 Asterinas 内核中实现了 IPv4 原始套接字（SOCK_RAW），支持 ICMP 协议的发送和接收。实现了 IP_HDRINCL 选项以允许用户控制 IP 头部，并正确计算 ICMP 校验和。该功能使得 ping 等网络诊断工具能够在 Asterinas 上正常运行。

**Linux Socket 语义兼容**：针对 Linux 网络编程中常见的语义进行了兼容性实现。包括：bind 到 0.0.0.0（INADDR_ANY）时作为通配符监听所有网络接口；TCP 监听套接字的通配符行为；UDP 子网广播投递支持；自动绑定（auto-binding）等。这些改进使得大量现有网络应用无需修改即可在 Asterinas 上运行。

**Netfilter / iptables 兼容层**：实现了完整的 IPv4 netfilter 钩子框架，支持 NF_ACCEPT、NF_DROP、NF_QUEUE 等标准裁决。在此基础上构建了 iptables-legacy 兼容层，支持 filter 表和 nat 表，包括：规则匹配（ICMP 类型匹配、网络接口匹配）、规则计数器（数据包和字节计数）、DNAT 目标、默认链策略（ACCEPT/DROP）、规则保存与恢复（iptables-save/iptables-restore）。该实现参考了 Linux netfilter 架构但进行了简化，适用于 Asterinas 内核的组件化设计。
