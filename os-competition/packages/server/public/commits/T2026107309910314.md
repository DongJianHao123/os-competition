#### 3.1.1 详细模块介绍
MATRIX 项目是一个 Linux 内核模块+用户态工具链，实现异构内存（DRAM+NVM）的冷热页面自动迁移。内核模块分为入口层（matrix_main）、基础设施层（utils、system_aware）、策略过滤层（process_grouping、repabp/monitored_group）、核心数据采集层（buffer_pool、bptree）、后台处理层（scan、optimize）和执行引擎层（action/swap）。用户态包含守护进程（matrix_user_daemon.py）、CLI 工具（matrix-user.py）、热度决策模块（decision.py）、sysfs 封装（repabp.py、system_aware.py、usysfs.py）以及压力测试工具（stress*.c、test_swap.c）。项目依赖内核补丁（matrix.patch）修改 swapfile.c、swap_slots.c、swap_state.c、vmscan.c 以支持定向换出。
