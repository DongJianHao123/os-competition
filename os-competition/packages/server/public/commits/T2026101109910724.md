#### 3.1.1 详细模块介绍
StrideANN 是一个面向大模型 Agent 长时记忆场景的外存向量检索系统原型，基于 DiskANN 的 SSD 常驻图索引，围绕图检索的随机 I/O 瓶颈进行系统级优化。系统拆分为三层：Base DiskANN 外存图索引（只读）、读路径流水线执行层（候选驱动异步流水线、动态宽度控制、邻接表复用）、动态写入覆盖层（WAL+Delta+Tombstone，将随机写转化为顺序日志追加）。此外还包含离线主动调参器，用于根据历史实验记录推荐流水线参数。项目参考了 Gorgeous 的页级布局和 PipeANN 的异步流水线思想，将布局局部性、运行时流水线调度、写入覆盖层和离线参数调优组合成 I/O 优化原型。
