#### 3.1.1 详细模块介绍
项目 OS_Kernel_Agent 是一个基于 LLM 的 OS 内核仓库自动分析与比较流水线系统。系统采用 A/B/C 三组协作架构：A 组负责静态分析和 Call Map 构建（接口a），B 组负责报告生成、FactChecker 校验和比较报告（接口b），C 组负责历史库构建、相似度计算、Pipeline 编排、后端 API 和前端。核心流水线通过 scripts/run_pipeline.py 编排，依次执行静态分析、代码指纹、相似度计算、报告生成、比较报告生成、FactChecker 校验、评估、导出和验证。系统支持规则模式和 LLM 增强模式，LLM 失败时可自动回退到规则模式。代码指纹采用 winnowing 算法，相似度计算综合六个维度加权。历史作品库已构建 60 个完整历史项目，并通过 leave-one-out 测试验证。
