#### 3.1.1 详细模块介绍
项目采用四层架构：交互层（ArkTS前端、Agent+MCP）、控制层（ROS2 Controllers、Robot State Nodes）、仿真层（Gazebo/MuJoCo仿真节点）、数据与模型层（LeRobot Data Collector、VLA Train&Inference）。核心模块包括：oh_agent（AI Agent层，集成LLM推理、MCP协议、任务调度）、control_svc（控制服务层，gRPC桥接ROS2控制）、controllers（控制器层，含agent2c、moveit2c、servo2c）、drivers/xl_chassis（底盘驱动层，HTTP API控制）、sim_drivers（仿真驱动层，MuJoCo集成）、demos（演示层，病房配送业务逻辑）。项目支持OpenHarmony原生适配、端侧伺服遥控、多模态具身任务、全流程模型训练。
