#### 3.1.1 详细模块介绍
项目AutoReKV-HL是一个面向资源受限大模型推理的运行时内存管理框架。核心设计分为三层：(1)运行时内存对象抽象层（core模块），将权重加载、KV缓存、专家缓存、激活工作空间和I/O开销统一为可观测、可迁移、可回放的内存对象；(2)在线联合内存控制层（controllers模块），包含Weight/KV/Expert三个控制器和GlobalMemoryArbiter全局仲裁器，在统一预算约束下选择策略；(3)离线回放自优化层（autorekv_hl + heuristic_learning + replay），通过回放环境、事后标注、策略归档和回归测试套件实现自我优化。项目代码组织清晰，模块划分合理，各模块间通过接口解耦。
