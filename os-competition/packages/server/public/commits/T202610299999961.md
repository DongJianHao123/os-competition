#### 3.1.1 详细模块介绍
项目基于 openvela 系统，通过 Git 分支管理多个版本的补丁包。主要开发内容为 flashinfo 示例，该示例在 apps/examples/ 下新建 flashinfo 目录，编写 flashinfo_main.c 程序，通过 ioctl 获取块设备几何信息。项目包含 V1 和 V2 两版补丁，V1 为基础版本，V2 增加了更完善的构建配置（Kconfig、Makefile、Make.defs）和板级配置（defconfig、启动脚本等）。团队还编写了自动应用脚本（apply-v1.sh、apply-v2.sh），支持先检查再应用补丁。
