#### 3.1.1 详细模块介绍
**传感器数据采集模块**：通过UART1（波特率9600, 8N1）读取SHT30温湿度传感器数据，使用环形缓冲区实现非阻塞读取，解析格式为"R:%fRH %fC\n"的帧，校验范围温度[-40,80]°C、湿度[0,100]%RH，连续10次未收到新帧则失效。光照传感器BH1750通过I2C0（地址0x23）读取，连续高分辨率模式，光照值计算为(buf[0]<<8|buf[1])/1.2，范围校验[0,65535]。

**风扇PWM调速控制模块**：在独立线程中以100Hz频率翻转FAN_SPEED_CONTROL GPIO，通过调节占空比实现无级调速，分辨率1%。GPIO低电平有效，占空比≤0%停转，≥100%全速，中间按比例翻转。线程安全通过互斥锁保护。

**GPIO驱动模块**：基于平台驱动模型，通过设备树解析子节点，为每个GPIO创建sysfs属性接口（direction、value、label），注册到自定义类"efx_gpios"下。兼容字符串"efinix,efx-gpio"。

**I2C驱动模块**：Efinix Sapphire SoC平台I2C总线驱动，支持标准（100kHz）、快速（400kHz）和快速+（1MHz）三种速率。通过MMIO访问寄存器，主模式支持重复起始条件，从模式通过中断状态机实现。兼容字符串"efinix,efx-i2c"。

**Qt图形界面模块**：采用事件驱动+单例模式架构，8大功能模块包括中央主控板（30fps指数平滑动画）、温度趋势图（Catmull-Rom样条+自适应加权预测）、外设控制面板、阈值设置面板、报警记录表格、光照趋势图、湿度趋势图、设备运行状态面板。支持6套内置配色方案（深色系列4套+浅色系列2套），21套静态QSS样式库，DPI自适应5档分辨率预设。

**网络通信模块**：UDP广播自动发现服务器（端口9999），TCP长连接（端口8080），JSON协议通信。上行数据格式{"type":"data","time":"HH:MM:SS.mmm","temp":"xx.x","hum":"xx.x","lux":123}，下行控制命令支持beep、led、fan等。断线自动重连，连续3次失败后重新UDP发现。

**数据存储与日志模块**：内存环形缓冲区2000条FIFO，批量50条写入SQLite WAL数据库，支持分页查询、CSV导入导出、统计信息。

**Buildroot系统集成模块**：外部包sensor_app集成，本地源码构建，SysV init脚本实现开机自启。设备树定制（linux.dts, uboot.dts, sapphire.dtsi），免登录自启动配置。
