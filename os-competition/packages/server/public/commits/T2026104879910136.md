#### 3.1.1 详细模块介绍
OS-Agent是一套以Claude Code Skill + MCP形式运行的内核作品查重和差异评审系统。系统核心功能包括：Git版本锁定、代码范围校验、指纹计算、1-vs-N搜索、函数匹配、关键证据锚定、完整性校验和HTML渲染。Claude Code负责阅读源码和文档，判断外部依赖、参考作品、实现度、原创度与关键差异。最终生成两份报告：report.html（面向评委的中文完整框架评审）和provenance.html（面向技术复核的函数溯源附录）。
