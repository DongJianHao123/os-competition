#### 3.1.1 详细模块介绍

**项目整体定位**：AgentMem是一个面向LLM智能体记忆场景的I/O优化向量检索引擎。项目针对智能体记忆访问的独特模式（recency-skewed读取、bursty写入、图遍历的空间局部性）进行了系统级优化。核心创新包括：(1) 图感知的磁盘布局（BFS/GOrder），将Vamana图的近邻节点在磁盘上连续排列，减少随机I/O；(2) 智能体感知的缓存预热，通过采样查询统计扇区访问频率，自动固定热点扇区；(3) 下一跳预取，在beam search过程中提前发起I/O；(4) 真实智能体记忆工作负载生成器；(5) LangChain集成，提供兼容LangChain的VectorStore接口。

**开发过程**：项目始于2026年6月9日的首次提交，包含基础DiskANN引擎框架（Vamana图、PQ、BufferPool、IO后端）。2026年6月18日第二次提交是核心创新提交，新增了graph-aware layout、agent-aware cache warmup、next-hop prefetch、C API、Python绑定、LangChain集成、工作负载生成器、完整评测脚本和结果图表。2026年6月29日第三次提交将README从英文重写为中文完整技术报告。三次提交时间跨度20天，演进清晰，从基础框架到创新特性再到文档完善，符合真实开发节奏。
