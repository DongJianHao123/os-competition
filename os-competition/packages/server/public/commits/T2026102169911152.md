#### 3.1.1 详细模块介绍
本项目是一个针对Anolis OS内核CVE的livepatch（热补丁）自动生成智能体。核心流程为：自动解析CVE → 获取补丁 → 分析补丁 → 检查目标内核 → 应用补丁 → 构建kpatch → 分类失败 → 重写重试 → 验证 → 生成报告。项目采用模块化架构，包含补丁重写引擎（RewriteAdvisor，支持6种策略轮转）、失败分类器（FailureClassifier，覆盖apply/build阶段多种失败模式）、RAG知识库（存储内核API迁移、构建失败案例等）、Web控制面板（可视化流水线、Docker集成）等。截至2026-06-30，已在4个适用CVE上成功生成livepatch.ko，patch应用成功率从33%提升至67%，kpatch编译成功率从0%提升至67%。
