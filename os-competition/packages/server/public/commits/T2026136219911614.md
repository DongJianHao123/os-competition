#### 3.1.1 详细模块介绍
项目为2026年全国大学生计算机系统能力大赛OS功能挑战赛道参赛作品，赛题为"面向具身智能的国产操作系统ACT模型适配与实时推理优化"。当前仅完成任务三（QEMU/StarryOS/riscv64），任务一（荔枝派SG2002 TPU）和任务二（香橙派RK3588 NPU）待开始。

任务三的核心工作是在StarryOS（RISC-V 64, QEMU虚拟机）上成功部署ACT模型CPU推理。技术栈包括：StarryOS、QEMU 10.2.1、RISC-V 64架构、ONNX Runtime 1.21.0（musl预编译版）、C语言推理程序、riscv64-linux-musl-gcc交叉编译、PyTorch 2.10.0导出ONNX opset 17。

部署流程：1) 使用export_onnx.py将PyTorch模型导出为ONNX；2) 使用prepare_inputs.py预处理输入数据（图像归一化、状态归一化）；3) 交叉编译act_onnx_c.c推理程序；4) 编译StarryOS并将推理程序、模型、输入文件注入rootfs；5) 启动QEMU运行推理。推理结果验证了左转和右转两个方向，输出shape为[1, 8, 3]。关键指标：推理程序13KB、ONNX模型193MB、运行时依赖库36MB、推理时长约215秒、运行时内存约230MB、StarryOS内核37MB、QEMU分配1GB内存。PyTorch对照验证显示最大误差0.000001。
