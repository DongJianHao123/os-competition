#### 3.1.1 详细模块介绍
VeriSpecOSLab 是一个面向操作系统课程教学的规约驱动开发与验证平台。项目采用 TypeScript/Bun monorepo 架构，核心模块 vos-core 实现了完整的 CLI 工具链，包含 40+ 子命令覆盖初始化、构建、测试、验证、Agent 协作、知识库管理等全流程。vos-agent 模块实现了 7 种 Agent 身份（spec-author、implementer、debugger、reviewer、reporter、toolchain-author、knowledgebase），通过能力包和 StageGate 梯度释放机制实现受控 AI 协作。vos-kb 基于 sqlite-vec 向量索引构建知识库，强制 Agent 回答附带来源引用。xv6-spec 参考项目是一个完整的规格驱动 RISC-V 教学内核，包含 67 个 OperationContract、21 个内核子模块、48 个公开测试，所有代码由 vos agent generate 从规格生成，并完成了九阶段全流程验证。
