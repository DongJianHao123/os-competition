#### 3.1.1 详细模块介绍
QuantumOS 采用分层架构设计，内核侧通过 `/dev/quantum` miscdevice 提供 ioctl ABI 接口，实现量子任务（Qernel）的完整生命周期管理。内核模块包含核心数据空间（KDS）、调度器、分配器、批处理引擎、前后端处理、状态管理、校准、控制、流水线编排等子模块，代码总量约 40 万行。用户态提供完整的命令行工具集（qrun/qstatus/qresult/qcancel/qresource/qversion/qfault/qrelease/qdaemon），以及 Python 工作进程（切割/执行/重构），支持从 QASM 输入到结果重构的端到端流程。测试脚本覆盖 Phase A/B/C 和 V1 多级测试体系。
