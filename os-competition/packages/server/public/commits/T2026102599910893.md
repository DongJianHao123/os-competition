#### 3.1.1 详细模块介绍
osAgent 是一个针对操作系统内核课程设计仓库的自动化分析平台。项目采用 Python 实现，包含完整的 CLI 工具链和 Web 可视化界面。核心功能包括：多仓库并发克隆与管理、L1/L2/L3 三层静态分析（快速概览→内核深度分析→跨仓库结构签名）、基于 RAG 的智能问答系统（检索事实表+DeepSeek LLM+抗幻觉验证）、单仓库/对比报告生成（支持 md/html/json 格式）。项目从 2026-06-30 11:40 开始初始化，到 23:53 完成最终文档更新，在一天内完成了从骨架搭建到功能完善的完整开发周期。
