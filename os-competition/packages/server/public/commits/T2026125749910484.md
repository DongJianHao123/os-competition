#### 3.1.1 详细模块介绍

**bci_core模块**：采用C语言多进程架构，由bci_master统一管理bci_eegd（采集）、bci_processed（处理）、bci_httpd（HTTP服务）、bci_recordd（记录）四个子进程。进程间通过POSIX共享内存（shm_open+mmap）通信，支持自动重启崩溃进程（最多3次）。bci_eegd支持三种数据源：模拟器（正弦波生成）、串口（TGAM/NeuroSky协议解析）、RPMsg（预留）。bci_processed实现C规则引擎情绪推理，并预留OP-TEE CA接口。bci_httpd提供20+ REST API端点，覆盖健康检查、EEG数据、情绪结果、YOLO检测、系统状态、TEE验证等。bci_recordd支持CSV和SQL journal两种持久化模式。

**brain_model模块**：基于PyTorch实现单通道EEG情绪四分类模型。v5模型采用Multi-scale 1D CNN + BiGRU + Attention架构，输入17维频带特征，输出happy/sad/angry/fearful四类概率。训练流程包括数据增强（噪声注入、幅度缩放、时间偏移、频带掩码）、SMOTE过采样、StratifiedKFold交叉验证、OneCycleLR调度、早停机制。v6模型进一步优化，进行32通道搜索和配置搜索（窗口大小、步长等超参数调优）。v6_emotion_proxy.py作为独立HTTP代理服务，加载训练好的PyTorch模型，替换C后端的情绪推理接口。

**eeg_cloud_server模块**：基于FastAPI构建，提供REST API和WebSocket实时推送。支持接收EEG数据、YOLO检测结果，内置规则引擎情绪推理（z-score模板匹配+EMA平滑），提供Web仪表盘页面。EEG_Bridge子模块包含C#蓝牙桥接程序和Python ThinkGear TLV解析器。

**web_client模块**：纯原生JavaScript单页应用，无框架依赖。实现EEG 8频带波形图（Canvas）、情绪概率柱状图、YOLO检测框叠加、系统状态面板、TEE信任验证面板、数据库记录浏览等功能。每700ms轮询12个API端点更新数据。

**ROS集成**：包含ros_astra_camera（Orbbec Astra相机驱动，支持多种型号）、yolo_detector（YOLOv8n目标检测，OpenVINO和NCNN格式）、ros_brain（EEG数据发布和YOLO结果桥接）。支持通过web_video_server提供MJPEG视频流。

**OP-TEE集成**：在飞腾派上部署OP-TEE 4.0，实现CA（bci_tee.c）和TA（bci_emotion_ta）通信。TA提供记录摘要生成和完整性验证命令。API端点/api/v1/system/tee/stats返回会话统计（session_count=72, error_count=0），/api/v1/system/tee/verify返回verified=true。平均延迟约260817μs。
