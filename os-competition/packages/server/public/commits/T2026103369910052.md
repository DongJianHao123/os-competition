#### 3.1.1 详细模块介绍
项目基于 Asterinas（星绽OS）——一个用 Rust 编写的 framekernel 操作系统，实现了完整的 virtio-gpu 全栈 GPU 加速通路。DRM Core 框架从零搭建，包括设备模型、IOCTL 分发、GEM 内存管理和 KMS 对象拓扑。SimpleDRM 驱动作为最小可行产品，通过 memfd 模拟显存打通了显示链路。随后逐步扩展 framebuffer 兼容层、KMS atomic modesetting 管线、virtio-gpu 设备驱动、VirGL 3D 加速和 Vulkan/Venus 支持。开发过程详细记录在 12 篇技术文档中，涵盖设计思路、实现细节、问题排查和验证效果。
