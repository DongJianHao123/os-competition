#### 3.1.1 详细模块介绍
**智能分析 Agent**：采用 LangChain4j 框架集成通义千问 qwen-plus 大模型，通过 ReAct 模式实现自主探索。支持 4 个工具（search_code/read_file/list_file_tree/get_build_config），最多 4 步循环。RAG 检索采用多查询重写（multi-query, 4 条）+ BM25/向量/RRF 融合 + 重排序（gte-rerank-v2）+ 调用图扩展，Hit@1 达 91.67%。支持分层记忆管理（project_knowledge 表、会话摘要、长期记忆跨对话复用），以及追问模式。

**多维比对引擎**：实现五层串行流水线——Layer1 静态量化+代码查重（Winnowing 指纹+类型感知词法归一化+多特征融合相似度），Layer2 AST 骨架指纹，Layer3 符号/调用图对齐+子系统差异（16 个子系统），Layer4 创新点分析 Agent，Layer5 事实核查 Agent。支持 xv6/rCore 基线三方扣除，倒排索引+LSH 候选预筛。

**历史作品库**：支持 Excel 批量导入 168 项历史作品信息，SimHash 反查 <20ms。支持即焚模式（使用后自动删除）、持久化指纹存储、按需 re-clone 历史仓库。指纹文件与主系统二进制兼容。

**抗幻觉机制**：Verifier-in-the-Loop 三段式架构——Drafter 生成初步报告，Verifier 自动扫描实际文件路径和代码内容验证引用，Renderer 渲染最终结果。模块文件全景定位实现文件级 Recall 100%。

**评委评分卡**：6 维优越性对比（架构/算法/工程/接口兼容性/创新/其他）+ 双方创新点 + 推荐评分，A4 单页一键 PDF/打印。

**离线 Skill 技能包**：os-plagiarism-mini，纯 Python 标准库实现，零第三方依赖。提供四个 CLI 命令：fp（指纹生成）、compare（1v1 比对）、build-lib（构建指纹库）、scan（1vN 反查）。采用 k-gram=5, window w=4, FNV-1a 64bit, SimHash 64bit 算法。

**基线演进图**：基于 SimHash 数据驱动，ECharts 力导向图自动识别 xv6/rCore/uCore/原创基线及年度迭代，共 141 节点谱系。
