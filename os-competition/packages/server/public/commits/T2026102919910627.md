#### 3.1.1 详细模块介绍
项目核心为"内核宕机Patch智能匹配系统"，通过解析Linux内核崩溃日志（Oops/Panic），提取关键错误信息和函数调用栈，在本地Linux内核Git仓库中搜索相关的upstream patch commit，并按相关性评分排序返回Top-N结果。系统支持CLI命令行模式和Web可视化界面两种使用方式，并提供Docker容器化部署方案。开发过程从2026年5月18日持续到6月29日，经历了日志解析、Git搜索、评分排序、Web界面、Docker部署等多个迭代阶段，最终在5个真实crash样本上达到Top-3命中率60%。
