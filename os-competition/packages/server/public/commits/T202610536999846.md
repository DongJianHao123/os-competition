#### 3.1.1 详细模块介绍
ZVM项目是一个基于Zephyr RTOS的Type-1 Hypervisor（虚拟机监视器），目标平台为飞腾E2000（ARM64架构）。项目核心代码位于zvm_e2000目录，整体架构遵循Zephyr RTOS的构建体系。项目包含完整的ARM64架构支持（启动、MMU、中断处理、SMP多核）、飞腾E2000Q板级支持包、以及大量设备驱动。项目还提供了预编译的固件镜像（zvm.elf）以及配套的FreeRTOS和Zephyr客户机镜像。此外，项目包含一个VisualZVM可视化工具（V2.0版本，约253MB的zip包），用于虚拟机管理。
