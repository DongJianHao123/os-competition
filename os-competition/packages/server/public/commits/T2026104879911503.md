#### 3.1.1 详细模块介绍
本项目基于llama.cpp开源项目进行二次开发，核心创新在于将LLM推理与WebAssembly沙箱技术结合。团队修改了WAMR运行时以支持更高效的WASM指令执行（如松弛指令、f16/f32转换），并开发了基于llama.cpp的对话应用。项目还包含一个完整的演示视频和演讲PPT，展示了在云边端场景下的智能负载运行效果。整体架构采用eBPF-based模块化运行时设计，优化了WASI接口。
