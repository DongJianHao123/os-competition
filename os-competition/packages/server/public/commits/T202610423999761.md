#### 3.1.1 详细模块介绍
PatchWeaver 是一个面向 Anolis OS ANCK 内核的 CVE livepatch 生成与验证 Agent 系统。用户输入 CVE 编号后，系统自动完成上游补丁获取、目标内核适配评估、语义等价改写、kpatch-build 构建、模块加载/卸载验证、功能与回归验证记录，以及结构化报告输出。系统采用 LangGraph StateGraph 实现 Agent 自治闭环，包含 observe、plan、guard、execute、reflect、reduce、route 等节点。Planner 只能调用注册的受控动作，Guard 负责安全审核，Reflection 将失败范式写入可复用记忆。生命周期采用有限状态机控制，关键迁移约束确保证据闭合后才能推进状态。系统支持 Web 控制台、命令行、API 服务和百炼应用四种使用方式。
