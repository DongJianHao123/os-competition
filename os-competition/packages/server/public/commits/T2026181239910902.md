#### 3.1.1 详细模块介绍
本项目在 Asterinas（Rust framekernel）上用 Rust 实现了兼容 POSIX 的 EXT4 文件系统。核心架构分为五层：应用/系统调用层、Asterinas VFS 集成层、EXT4 核心语义层、JBD2 一致性层、块设备适配层。支持 Extent 树管理、JBD2 ordered 日志、PageCache 缓存、mmap 映射、O_DIRECT 直接 I/O、多线程并发读写。开发过程从 2026-06-01 的 stage1 基础支持开始，经过 8 个 stage 逐步完善，再进入 JBD phase1-4 的日志集成，最后进行性能优化和比赛材料准备。性能方面，SQLite speedtest1 从 ~2022s 优化至 ~234.9s（8.6x 提升），fio O_DIRECT 顺序 I/O 达到 Linux EXT4 的 75%-123%。
