#### 3.1.1 详细模块介绍
AhRightOS 是一个完整的操作系统教学实验平台，基于 Rust/RISC-V 64 架构，运行在 QEMU virt 机器上。项目从 ErrorOS 原始教学内核出发，进行了全面的组件化重构和平台化扩展。内核层面，将内存管理、进程管理、文件系统拆分为独立 crate，支持宿主机单元测试，并新增了事件日志系统、VMRegion、ELF 安全校验等功能。教学平台层面，构建了 PTA 后端（FastAPI）和前端（Next.js），提供实验指导书阅读、Lab 在线编辑与测评、QEMU 事件可视化、Rust 教程 Notebook 和伴学 Agent 等功能。课程工程层面，建立了完整的文档中心（分区管理）、ADR 架构决策记录、任务验收清单、课程 profile 和合同测试体系。项目还引入了 Linux-inspired 的虚拟文件系统概念（procfs、sysfs、devtmpfs）和 Component Platform 组件化架构。
