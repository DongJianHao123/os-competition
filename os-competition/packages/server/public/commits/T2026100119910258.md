#### 3.1.1 详细模块介绍
项目基于 rCore Tutorial 组件化操作系统教学实验教程，完成了赛题要求的全部5个基础实验练习（ch3 sys_trace、ch4 mmap/munmap、ch5 spawn/stride 调度、ch6 linkat/unlinkat/fstat、ch8 银行家算法死锁检测），并在此基础上设计了个性化教学实验环境。团队采用分工协作模式：同学 A 负责 ch3/ch4/ch5，同学 B 负责 ch6/ch8。开发环境基于 CNB 云平台，使用 VS Code Web IDE + Claude Code CLI 进行 AI 协作开发。项目还新增了第9章 VFS 扩展和 ch8-skeleton 对抗式 AI 教学练习骨架，体现了较强的创新性。
