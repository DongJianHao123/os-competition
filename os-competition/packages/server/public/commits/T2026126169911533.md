#### 3.1.1 详细模块介绍
本项目基于 Asterinas（星绽）开源操作系统内核（https://github.com/asterinas/asterinas），在其 framekernel 架构上实现了 ptrace 进程调试子系统。Asterinas 是一个 Linux ABI 兼容的 Rust 操作系统内核，采用 framekernel 架构将内核分为安全上层（kernel/，纯安全 Rust）和不安全下层（ostd/，包含 unsafe 代码）。

团队在原始代码基础上新增了以下 ptrace 相关功能：
1. **PTRACE_ATTACH/DETACH**：支持调试器附加到目标进程和分离，包含权限检查、Yama 安全模块集成。
2. **PTRACE_GETFPREGS/SETFPREGS**：读写浮点寄存器状态，通过 FpuContext 字节拷贝实现。
3. **PTRACE_GETFPXREGS/SETFPXREGS**：扩展浮点寄存器支持，x86_64 上 SETFPXREGS 返回 EINVAL（符合 Linux 行为）。
4. **PTRACE_SYSEMU/SYSEMU_SINGLESTEP**：系统调用模拟执行，在 syscall 入口处拦截并跳过实际执行，由 tracer 模拟返回值。
5. **PTRACE_GETREGSET/SETREGSET**：现代 iovec 接口的寄存器读写，支持 NT_PRSTATUS 和 NT_PRFPREG 类型。
6. **x86_64 寄存器 ABI**：完整的 Linux 兼容寄存器布局和访问策略，包括通用寄存器、段寄存器、标志寄存器、浮点寄存器等。
