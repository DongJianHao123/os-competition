#### 3.1.1 详细模块介绍
项目核心为 KV Cache 分层管理系统，通过 PARS（Predictive Adaptive Reuse and Stall-aware）算法将 KV 块按热度分配到 GPU（热层）、CPU（温层）和磁盘（冷层）。系统包含确定性模拟器用于算法验证，HTTP Sidecar 提供外部集成，vLLM Hook/Bridge 实现与主流推理框架的集成，以及完整的测试、审计和证据收集工具链。开发过程从 2026-05-31 开始，经历了 50 次提交迭代，从初始原型逐步演进到完整的系统原型。
