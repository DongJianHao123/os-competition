#### 3.1.1 详细模块介绍
项目以 xv6-riscv 为基线，采用"干净基线 + 补丁"的工作流，不提交第三方源码。每个实验模块对应一个或多个补丁文件，配合测试程序和验证脚本。Lab0 提供环境搭建和基线验证；Lab1 通过 hello 和 add2 教学系统调用完整路径；Lab2 通过 pstate/pcount/pchildtest 观察进程状态；Lab3 通过 pgcount/memstat 观察页表映射；Lab4 通过 fcount/fdcount/fdinfo 观察文件表和 fd 表；Lab5 为综合复现实验。集成补丁 0001-0009 将所有功能合并，支持一键验证。所有补丁均从上游 xv6-riscv 基线生成，可追溯。
