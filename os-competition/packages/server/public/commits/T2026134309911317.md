#### 3.1.1 详细模块介绍
本项目基于 xv6-riscv-rev5，实现了虚拟文件系统（VFS）层和块设备 HAL，使内核能同时支持 xv6-fs、ext2 和 FAT32 三种文件系统。VFS 层提供了通用的 inode 操作接口、超级块管理、挂载表、路径解析等功能。块设备 HAL 将多个 virtio-blk 实例抽象为统一接口，支持多磁盘路由。ext2 驱动实现了完整的 ext2 文件系统读写，支持 1024 字节块大小、直接/间接块寻址。FAT32 驱动实现了完整的 FAT32 文件系统读写，并维护 FSInfo 以确保 fsck.fat 检查通过。所有驱动均通过 VFS 层注册，用户可通过 mount/umount 命令动态挂载/卸载文件系统。
