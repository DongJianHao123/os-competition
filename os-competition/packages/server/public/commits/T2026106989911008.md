#### 3.1.1 详细模块介绍
oskag 是一个面向小型操作系统内核的代码分析比对智能体系统，采用四阶段流水线架构：facts 提取 → describe 模块证据 → verifier 验证 → render 渲染。系统支持对 ArceOS/Starry、rCore/Rust、xv6/C、Unix/C 等五类内核家族进行自动识别和分析，具备三闸防幻觉机制（引用验证、语义验证、交叉验证），多路径相似度评分，以及定量评估指标（引用精确率 99.24%，家族检测命中率 86%）。开发团队为精勤求学（西安交通大学）和 xuejun，提交记录从 2026-06-23 持续到 2026-06-29，共 20 次提交，开发活跃且持续演进。
