#### 3.1.1 详细模块介绍
项目基于 OpenHarmony Mini（LiteOS-M）轻量级实时内核，依托 WSL2 + QEMU ARM 虚拟仿真环境，实现了操作系统核心原理系列教学实验。实验分为三层：基础准备层（实验一、二）、内核核心机制层（实验三~五）、系统综合应用层（实验六~八）。每个实验均提供 Shell 自定义命令进行交互。此外，配套自研 Web 教学平台（Go/Gin 后端 + Vue3/TypeScript 前端 + SQLite 数据库），支持学生提交 PDF 实验报告、AI 自动评分（基于大模型，按实验设计/数据分析/结果讨论/报告规范四维度评分）、教师管理成绩、管理员配置系统。Web 平台已部署公网可访问（https://liteos.0pt.icu）。
