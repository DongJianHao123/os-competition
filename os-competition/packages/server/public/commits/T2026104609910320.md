#### 3.1.1 详细模块介绍
项目采用 eBPF 技术实现非侵入式系统级异常监控。内核态通过 BPF 程序挂载到 tracepoint、kprobe、uprobe、LSM 等钩子点，采集进程、文件、网络、SSL 等事件，通过 Ring Buffer 传递到用户态。用户态程序（ai_monitor.c）负责事件处理、Agent 状态管理、行为熵分析、告警生成等。项目还实现了滑动窗口限速、行为基线偏差检测、容器 ID 提取、热更新配置等高级功能。测试覆盖单元测试、集成测试、功能测试、安全测试和性能测试，共 136 个子测试全部通过。
