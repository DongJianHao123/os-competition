#### 3.1.1 详细模块介绍
项目基于 TanGram-rCore-Tutorial 参考环境进行改良，采用组件化 crate 结构组织代码。每个章节（ch1-ch5）为独立的内核 crate，运行在 RISC-V 64 架构 QEMU 虚拟机上。团队原创内容包括：ch2 改进 Trap 错误信息（从单行输出扩展为异常类型、触发地址、指令地址、可能原因四行详细输出）；ch3 实现 sys_trace 系统调用（三种 trace_request：读取用户内存、写入用户内存、查询 syscall 次数），并在 TaskControlBlock 中新增 syscall_times 字段；ch4 实现 mmap/munmap 系统调用（地址对齐检查、权限合法性检查、避免重复映射、页表映射/取消映射），以及基于页表的安全 trace；独立 crate tg-rcore-trace 已发布至 crates.io，提供 SyscallCounter、TraceRequest、handle_trace() 等 API，含 5 个单元测试全部通过。项目还包含完整的测试框架（checker）和 50+ 用户态测试程序。
