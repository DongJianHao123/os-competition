#### 3.1.1 详细模块介绍
项目采用Rust语言开发，模块化设计，面向飞腾D3000M（ARM64/aarch64）平台，依赖axvisor虚拟化平台和StarryOS监控内核。监控模块通过三重Watchdog（硬件定时器、心跳、内存扫描）实现冗余崩溃检测，状态机驱动（Initializing→Active→Suspected→CrashConfirmed→Collecting→Completed→Error）。信息采集模块采用8阶段流水线（暂停Guest→寄存器采集→内存转储→内核日志→模块信息→栈展开→进程信息→持久化），支持四种内存转储策略。语义恢复模块实现三级栈展开退避（DWARF CFI→FP-based→Scan-based）和根因推理引擎，支持多格式报告生成。hypervisor模块定义了完整的Hypercall接口（12个Hypercall），通过共享内存环形缓冲区实现通信。
