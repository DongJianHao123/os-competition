#### 3.1.1 详细模块介绍
项目基于 uCore RISC-V 教学操作系统，扩展了内核级 AI Agent/LLM 工作流支持。核心创新在 os/agent.c（211KB）中实现，包括 Agent 进程标识与角色绑定、结构化工具调用协议（batch agent_run 支持最多 64 个操作）、可信 Context Path（128 条记录环形缓冲区，带因果链和哈希完整性保护）、文件元数据服务（128 条表项，关联真实 inode，支持索引/扫描双路径查询）、事件驱动 Agent 循环（16 槽事件队列，watch/wait/wake/cancel/heartbeat）、Agent 感知调度器（角色权重、可配置策略、事件优先级、deadline）、全局审计环形缓冲区（512 条记录，哈希链）、统一时间线导出、文件编辑租约（排他非等待，版本检查，直接写入拦截）、以及 LLM Relay 集成。项目采用双目标设计（plain uCore vs AgentOS-uCore），通过共享状态文件名和复核入口，对比内核支持前后的差异。用户态测试程序覆盖所有功能模块，包括 agentfinal_ucore（综合正确性）、agentfs_ucore（文件系统）、agentscan_ucore（自动扫描）、agentloop_ucore（事件循环）、agentsched_ucore（调度）、agentconflict_ucore（文件冲突）、agentllm_ucore（LLM Relay）、agentbench_ucore（性能基准）、labdemo_ucore（多 Agent 恢复场景）、agentsecurity_ucore（安全权限）等。
