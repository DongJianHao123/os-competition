#### 3.1.1 详细模块介绍

**方案一（oh_robot_sim）**：由钱欣萍（组长）主导开发，基于ROS2 Humble + Gazebo Classic 11仿真平台，实现了一套完整的视觉导航竞赛控制器。核心代码competition_controller.py实现了多阶段状态机（初始跟踪→仓库阶段→卸货阶段→拾取序列→回家阶段），通过HSV颜色空间检测赛道轨迹线（白色/灰色）、黄灯（黄色区域）、箭头（绿色区域）、终端（蓝色区域），使用pyzbar解码QR码获取任务信息，基于深度图像检测障碍物。独立模块lane_keeper.py实现赛道保持算法，detect_all.py集成YOLO物料检测+颜色检测+QR码检测。方案一使用Panda原生机械臂（静态展示）。

**方案二（xutianyang_handoff）**：由徐天炀开发，采用路径点P控制器导航+自制6-DOF机械臂。auto_nav.py实现WaypointFollower节点，以50Hz频率运行PID控制（kp=1.5, ki=0.05, kd=0.03），包含自适应漂移补偿机制。pick_object.py实现5步抓取流程（HOME→刹车→伸向箱子→对称闭合夹爪→提升）。gen_combined.py生成自制机械臂URDF。方案二解决了车身旋转、手指滑脱等稳定性问题。

**赛道模型**：李怡宁负责搭建了丰富的Gazebo模型库，包括breakable_test、bus、cabinet、cafe、cafe_table、charging_station、competition_road、material_A1-A4、person_standing、stop_light、suv、task_terminal、unloading_zone、yellow_light等20余种模型，以及origin.world世界文件。

**赛道地图**：钱欣萍使用FreeCAD设计了多版本赛道地图，从veryclose（6月11日）→round→haveroad→whole_road（6月15日）→whole_road_backup（6月26日），迭代过程清晰，有备份文件佐证。
