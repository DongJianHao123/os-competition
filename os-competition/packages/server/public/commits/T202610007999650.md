#### 3.1.1 详细模块介绍
Agent JS 项目采用 Rust 从零实现了一个轻量级 JavaScript 执行引擎，面向 AI Agent 的短时高频脚本执行场景。项目包含完整的词法分析器（lexer.rs）、语法分析器（parser.rs）、抽象语法树（ast.rs）、树遍历解释器（interpreter.rs）、内置对象（builtins.rs）、值系统（value.rs）等核心模块，以及可选的字节码编译器（compiler.rs）和虚拟机（vm.rs）性能路径。

开发过程从 2026 年 6 月 11 日的 base 提交开始，逐步实现了函数声明、闭包、控制流、对象/数组字面量、switch 语句、do-while 循环、for-in/for-of 循环、类语法、解构赋值、箭头函数、Symbol 支持、async 能力、DataView 支持等 ECMAScript 特性。test262 运行器支持分类解析/运行时/断言/超时/不支持失败，核心兼容矩阵稳定超过 80%。
