#### 3.1.1 详细模块介绍
**词法分析器 (lexer.rs)**：实现了完整的 JavaScript 词法分析，支持 ES2020+ 特性（可选链 ?.、空值合并 ??、逻辑赋值 ||= &&= ??=、私有标识符 #、静态块等）。支持 Unicode 标识符、模板字面量嵌套、正则字面量（含 u 标志验证）、BigInt、数字分隔符等。错误处理完善，使用 Result 返回带行号/列号的错误信息。代码结构清晰，方法职责单一。

**语法分析器 (parser.rs)**：递归下降解析器，覆盖大多数 JavaScript 语句和表达式。支持 ES2023 语法：类（含字段、静态块、getter/setter、私有字段）、箭头函数、async/await、生成器、模块导入导出、解构赋值等。实现了自动分号插入（ASI）、严格模式检查、标签语句验证等。代码使用 Rust 枚举和模式匹配，类型安全。

**AST 定义 (ast.rs)**：使用 Rust 结构体和枚举定义 AST 节点，通过 serde 序列化，兼容 ESTree 格式。Stmt 枚举覆盖 26 种语句类型，Expr 枚举覆盖 25 种表达式类型，Pattern 枚举支持 7 种模式。节点字段使用 Box 智能指针实现递归结构。

**解释器 (interpreter.rs)**：树遍历解释器，直接执行 AST。实现完整的作用域链与闭包机制，支持 this 绑定、super 访问、原型链查找。内置 Promise 微任务队列，支持异步执行和严格模式。代码量约 37 万行，是项目核心模块。

**对象系统 (object.rs)**：基于 Rust 的 Rc<RefCell> 实现共享所有权，支持原型链继承与属性描述符，实现 getter/setter 访问器属性，支持 Object.freeze、Object.seal 等对象操作。

**内置对象模块 (built_ins/)**：模块化设计，每种内置对象独立管理。Array 实现 30+ 原型方法，String 实现 40+ 原型方法，Object 实现静态方法，Math 实现数学函数，Date 实现日期操作，RegExp 实现正则匹配，JSON 实现序列化与反序列化。

**Test262 测试运行器 (test262-runner.rs)**：可运行 ECMAScript Test Suite 中 language 和 built-ins 测试，当前通过率约 60%。
