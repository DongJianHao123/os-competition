#### 3.1.1 详细模块介绍
项目是一个面向 OS 内核代码评审的 AI Agent，提供三条独立分析链路：
1. **描述报告链路**：通过 oskernel_agent 模块，基于 MCP 协议与 OpenCode 集成，对单个仓库进行自底向上静态分析，产出树状 HTML 报告。核心引擎支持 rust-analyzer LSP（高精度）、LLM 批量处理和路径规划三种模式。
2. **查重对比报告链路**：通过 embed（向量嵌入 + FAISS）、simhash（粗筛）、exact（精确匹配）、report/semantic_compare（2734 行核心语义比较）等模块，实现语义级查重，识别改名复制、跨文件挪用等，并扣除库复用、公共样板、基线衍生、误报等排除项。
3. **AI 生成代码检测链路**：通过 ai_detect 模块，使用免训练的 log-rank + NPR 两阶段信号检测，标记疑似 AI 生成代码段。

前端控制台（Vue + Node.js）支持导入作品清单、发起报告生成、查看日志和打开报告。后端通过 PipelineQueue 管理异步任务队列，支持三种报告类型的生成。
