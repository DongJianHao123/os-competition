#### 3.1.1 详细模块介绍
项目围绕赛题 proj61 "面向 AI 智能体的操作系统内核（Agent-OS）"，在 uCore/RISC-V64/QEMU 教学操作系统内核基础上进行扩展。内核侧实现了 Agent 身份管理、结构化工具调用（tool.c/h）、上下文路径（context.c/h）、事件驱动 Agent Loop（loop.c/h）、Provider frame（provider.c/h）、治理审计（audit.c/h、policy.c/h）和文件属性索引（sfs_attr.c/h）。用户态侧实现了 deterministic Planner Runtime（agent_planner.c/h）、MCP-style Provider adapter（agent_mcp_adapter.c/h、agent_mcp_bridge.c）和 host 证据 harness（agentos_harness.py）。项目还包含大量用户态测试程序（约50个.c文件），覆盖各模块功能验证。开发过程遵循模块划分与并行开发契约，使用施工图和 reference evidence 管控边界。
