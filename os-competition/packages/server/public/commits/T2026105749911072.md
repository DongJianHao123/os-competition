#### 3.1.1 详细模块介绍
项目实现了从P0到Lab7共8个递进式教学实验模块。P0建立了Rust no_std内核在RISC-V 64 QEMU/OpenSBI上的最小运行环境，包含SBI控制台输出和构建测试框架。Lab1在P0基础上实现启动链和测试标记输出。Lab2引入S-mode trap处理，实现断点异常捕获和寄存器保存恢复。Lab3实现物理页帧分配器，支持分配、释放和双释检测。Lab4实现Sv39三级页表恒等映射，支持map/unmap/translate操作。Lab5实现单hart内核态协作式轮转调度，包含任务状态机和上下文切换汇编。Lab6实现用户态进入和系统调用分发，支持write/exit/yield/read/open/close共6个系统调用。Lab7实现内存设备抽象和简化文件系统，支持文件打开、读写、关闭操作。每个模块均提供starter（学生起点）和solution（参考实现）两个分支，并有对应的自动化测试脚本。
