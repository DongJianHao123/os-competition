#### 3.1.1 详细模块介绍
**Task_1 系统移植**：项目基于OpenHarmony 5.0.3标准系统，针对启智01开发者套件（昇腾310B）进行了完整的系统移植适配。内核配置文件 ascend310B_oh_defconfig 共7813行，启用了Ascend特性集（CONFIG_ASCEND_FEATURES）、DRM显示驱动（CONFIG_DRM_HISI_ASCEND310B）、MMC控制器（CONFIG_MMC_SDHCI_ZODIAC_310B）、HDF驱动框架、Binder IPC、HiLog日志等关键功能。设备树补丁（6个）修改了bootargs以支持OpenHarmony启动、GPU中断、HDC USB、8T设备、音频和ISP摄像头。OpenHarmony补丁（44个）覆盖了编译白名单、HDF驱动、图形GPU、EGL、音频、摄像头、多屏显示、SELinux、HDC等多个子系统。

**Task_3 NPU推理应用**：YOLOv5推理程序基于CANN ACL运行时，支持OM模型推理，兼容YOLOv5u/YOLOv8（84x8400）和传统YOLOv5两种输出格式。包含完整的图像预处理（letterbox缩放、BGR转RGB、归一化）、后处理（解码、NMS）、检测框绘制和结果保存功能。推理服务器基于HTTP提供健康检查和推理接口。OpenHarmony图形应用使用ArkTS框架，包含AppScope、entry模块和hvigor构建配置，但native NAPI模块标记为待实现。
