#### 3.1.1 详细模块介绍
OSLearn采用"Web平台 + 数据层 + 异步Worker + 沙箱Runner + 诊断与教学分析"的分层架构。Web层提供两套入口（内置HTTP Server和FastAPI），数据层支持SQLite/PostgreSQL双后端，Worker层异步消费提交任务，Runner层为核心执行引擎。诊断与教学分析包括规则诊断、实验指纹、课堂案例库、诊断图谱、教师讲评包等。智能扩展通过contracts.py和extensions.py提供osRAG、负载自适应调度、学习画像、L0-L3分层提示等provider接口。
