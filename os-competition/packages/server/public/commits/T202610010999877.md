#### 3.1.1 详细模块介绍

**异常检测模块**：实现了LSTM-VAE和LSTM-AE两种深度学习异常检测器。LSTM-VAE编码器输出μ和logσ²，通过重参数化采样得到潜在变量z，解码器从z重建序列。损失函数为重构误差（MSE）加上β倍的KL散度。异常分数综合重构误差和KL偏移。支持自适应contamination、early stopping、模型保存与加载。predict方法包含窗口对齐、z-score归一化、中值平滑、阈值判断和段级后处理。

**GNN根因定位模块**：采用关系感知图神经网络架构（基于GMT-Flaky论文改进），包含2层GATConv图编码器（含残差连接）、RelationAwareEdgeScorer（拼接源/目标节点嵌入与关系类型嵌入后经MLP输出边评分）、Gumbel-Softmax离散子图采样（训练时）或Top-K边选择（推理时）、全局读出（Top-K注意力池化或Soft注意力池化）、多标签分类器（输出15类事件概率）。损失函数包含分类损失+紧凑性损失+监督对比损失。训练器使用AdamW优化器、ReduceLROnPlateau调度器、梯度裁剪，基于验证F1保存最佳模型。

**LLM可解释性模块**：CausalChainBuilder从GNN的edge_importance和因果图adj矩阵中构建因果传播链路，支持BFS反向追溯上游指标。NaturalLanguageExplainer优先尝试Ollama qwen2.5:7b模型动态生成自然语言解释，失败时降级到15类硬编码模板。包含few-shot示例懒加载、ChatML格式prompt构建、因果链ASCII图生成等功能。
