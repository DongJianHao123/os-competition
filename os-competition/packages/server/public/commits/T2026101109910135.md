#### 3.1.1 详细模块介绍
项目 README.md 包含两个主要项目的混合内容：
1. **ROS2/DDS 调优与验证**：面向操作系统设计赛 OS 功能挑战赛道，核心目标是优化 ROS2 DDS 实时性。文档详细描述了硬件基础设施（Ubuntu 22.04 LTS，Node A/B 网线直连）、基线测试、DDS 中间件调优（QoS 策略定制、SHM 传输优化、资源约束 CPU≤30% 内存≤128MB）、OS 系统级优化（多 DDS 横向对比、PREEMPT_RT 内核、网络抗干扰、动态 QoS）、工程验证与数据输出。包含 /odom 和 /camera/image_raw 话题的传输层测试结果。
2. **KernelInsight**：面向操作系统内核赛道作品的仓库智能分析与历史对比系统。功能包括自动导入 Git 仓库、分析源码结构、AI 辅助理解、构建仓库画像、多维度评分（原创性、新颖性、可实践性、技术难度、完成度）、生成 Markdown 报告等。当前只提交了后端算法部分。
3. **实时性能测试探针**：bench/ 目录包含 RROS、Xenomai、PREEMPT_RT 的实时延迟对比探针源码，用于飞腾派 V2.1 实测。
