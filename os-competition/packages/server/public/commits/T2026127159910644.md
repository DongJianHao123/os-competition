#### 3.1.1 详细模块介绍
本项目基于 Efinix Sapphire SoC 开发板（WZZY_TJ375N529，Buildroot Linux），构建了一套多传感器环境监测预警系统。系统采用四层分层架构：硬件采集层（串口温湿度传感器 + I2C 光照传感器）、嵌入式 Linux 底层（设备驱动与自启动脚本）、数据服务层（sensor_server.py 提供 HTTP JSON 接口）、PC 上位机应用层（PyCharm 工程，含 main.py 图形界面、config.json 配置、CSV 日志存储）。核心功能已全部调试落地并通过 72 小时压力测试，智能预警支持三级状态（温度≥32℃/湿度≥80%RH/光照≥7000 lux 触发红色预警）。散热风扇未装配，声光预警硬件仅预留 GPIO 接口待完善。
