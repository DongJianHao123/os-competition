#### 3.1.1 详细模块介绍
zeroTrace 是一个轻量级 Linux 用户态动态探针工具，使用 ptrace 向目标进程注入 payload，通过重写目标函数入口来收集参数、捕获返回值，完全在用户空间完成事件记录，避免每次探针命中时陷入内核。项目支持 x86_64 和 aarch64 双架构，提供交互式 CLI 进行探针管理，支持条件过滤、探针热更新、多线程安全等高级特性。项目包含完整的测试套件和基准测试脚本，文档齐全（含技术方案、章程、项目文档、汇报PPT）。
