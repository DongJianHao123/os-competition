#### 3.1.1 详细模块介绍
项目基于StarryOS/ArceOS内核框架，在Rockchip RK3588平台上进行AIOS设计与优化。团队核心工作包括：1）内核层面：SMP调度优化（round-robin跨核分发、waker回home CPU），提升多核利用率；2）NPU推理优化：通过日志门控降低串口IO对推理的干扰，通过SCMI协议在推理前提升CPU频率至2.4GHz；3）性能诊断：构建系统级profiler，定位ioctl瓶颈，实现跨层瓶颈分析；4）模型链路：将网球检测模型从ONNX转换为RKNN格式，采用hybrid INT8量化（4.25M），cos_sim达0.999263；5）上板bringup：完成SD卡启动、ums传输、NPU端到端推理与真图检测验证；6）自研工具：开发npu_smoke.c、pthread_smoke.c、pacer.py等辅助工具。
