#### 3.1.1 详细模块介绍

**Raw Socket 模块**：项目实现了 IPv4 raw ICMP socket 的最小可用子集。核心代码分布在 `kernel/src/net/socket/ip/raw.rs`（RawSocket 主体实现）、`kernel/libs/aster-bigtcp/src/socket/raw_ip.rs`（接收/发送队列和资源边界控制）等文件中。支持创建 raw socket 时的 CAP_NET_RAW 权限检查、完整 IPv4 packet 接收、ICMP Echo Request/Reply 收发闭环、loopback 场景下 ping 命令级验证、IP_HDRINCL 基础兼容路径、非阻塞读写和 poll readiness，以及对 raw packet 队列设置 packet 数量和字节数上限。Git 历史显示该模块在 6月21日-30日期间持续开发，commit 0beb91b 接入 IPv4 收发路径 hook，commit ffc896e 增加了 bind/connect 支持和 checksum 修复。

**Socket 语义兼容模块**：以 Flask、Python HTTP server、Redis 等常见服务的运行路径为目标，系统化修复了 Asterinas 与 Linux 在 IPv4 socket 服务端语义上的关键差异。主要修复包括：支持 TCP/UDP bind(0.0.0.0) 作为 INADDR_ANY 通配绑定、getsockname() 保留用户可见端点、未显式 bind() 的 TCP socket 直接 listen() 自动绑定、保持 127.0.0.1 loopback 路径正确、支持同一 listener 顺序 accept() 多个连接、补齐 poll/select/epoll readiness 语义、支持 SO_REUSEADDR 下快速重启、提升默认 socket buffer、对 AF_INET6 明确返回 EAFNOSUPPORT。核心代码在 `kernel/src/net/socket/ip/common.rs`、`addr.rs`、`stream/init.rs`、`stream/listen.rs` 等文件中。

**netfilter/iptables 模块**：从零实现了面向 Asterinas IPv4 数据路径的 netfilter/iptables 最小可用子集。定义了 PREROUTING、LOCAL_IN、LOCAL_OUT、POSTROUTING 四个 hook 点，支持 filter 表 OUTPUT 链、规则有序匹配 first-match 语义和默认 ACCEPT 策略，支持 ICMP/TCP/UDP 协议匹配、IPv4 地址匹配、端口匹配和 ICMP identifier 匹配，支持 ACCEPT/DROP target 和规则 packet/byte counters。提供 /proc/netfilter_rules 控制面和最小用户态 iptables shim。核心代码在 `kernel/libs/aster-bigtcp/src/netfilter/` 目录下（hook.rs、rule.rs、chain.rs、table.rs）。Git 历史显示从 6月21日（commit 6317acc 挂载 procfs 入口）到 6月25日（commit 0beb91b 接入 IPv4 收发路径 hook）持续开发，共涉及 10+ 个 commit。

### 4.2 遭遇的代码问题
| 问题描述 | 解决方法 |
| --- | --- |
| smoltcp 与 Linux 语义差异：smoltcp 面向轻量级嵌入式场景，不完整复刻 Linux 网络语义 | 将 Linux socket ABI 语义和 smoltcp 内部网络模型解耦，上层保持 Linux 用户态可见行为，下层仍按具体 iface 和具体 IPv4 地址完成收发 |
| 缺少 netfilter 框架：Asterinas 原有网络栈没有包过滤/NAT 框架 | 从零实现 netfilter hook 点、规则链、匹配器和 target，并接入 IPv4 数据路径 |
| IPv6 未完整实现 | 对 IPv4-only 路径中的 AF_INET6 明确返回 EAFNOSUPPORT，避免伪装支持 IPv6 |
| 资源边界控制：raw packet 可能无界占用资源 | 对 raw packet 接收/发送队列设置 packet 数量和字节数上限 |
| raw socket checksum 问题：iputils ping 报告 bad checksum | 使用默认 ChecksumCapabilities 处理用户可见包的 checksum |
