#### 3.1.1 详细模块介绍
NodeFusion 采用"三层协同"架构：工程内核层（xv6 Kernel Base）完整保留 xv6-riscv 核心模块，包括 Trap/系统调用路径、进程管理（PCB、调度）、内存管理（页表、缺页处理）、文件系统（inode、日志、VirtIO 驱动）等；教学控制层核心是自主设计的 LAB_STAGE 机制，通过编译期宏定义与运行期检查将实验阶段与内核能力显式绑定，实现闯关式教学模式；实验交付与评估层采用教师答案版/学生挖空版双版本工程结构，并集成自动化评测（Autograder），与 LAB_STAGE 深度协同。开发周期 2025.9.27-2025.12.30，分四个阶段完成全部 Lab 0-5 的开发与测试。
