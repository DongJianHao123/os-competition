#### 3.1.1 详细模块介绍
LUDP 项目实现了完整的用户态动态探针系统。核心模块 F1 通过 ptrace 附加目标进程，解析 /proc/<pid>/maps 获取动态库加载基址，解析 ELF 符号表计算函数运行时地址，然后将目标函数入口改写为 5 字节 jmp rel32 跳转到用户态 trampoline。探针命中时完全在用户态执行，不陷入内核。F2/F3 在 trampoline 中采集参数和返回值。F4 确保探针清理时完整恢复。F5 支持运行时动态开关。F6 支持最多 16 个探针共存。F7 通过暂停线程保证多线程安全。A1-A5 为扩展功能，包括 ARM64 后端（部分完成）、条件过滤、perf-script 事件输出、远程调用和过滤条件热更新。项目文档体系完整，包含架构文档（A1-A5）、功能文档（F1-F7）、技术设计文档、项目报告、基准测试报告和交互式演示页面。
