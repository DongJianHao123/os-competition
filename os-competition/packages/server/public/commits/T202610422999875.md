#### 3.1.1 详细模块介绍
项目围绕非易失性内存（NVM）写扰动问题，构建了一个基于强化学习（RL）的可复现优化框架。核心思路是将stale block bucket选择建模为在线决策问题，通过DQN系列算法在workload stream上训练RL策略，并与random、oracle、learnwd、pnw等baseline进行对比评估。项目包含完整的数据生成、环境建模、RL训练、评估和实验流水线，支持多种编码方案（raw/din/dmpart/minwd）和真实数据集。
