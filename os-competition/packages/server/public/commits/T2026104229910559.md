#### 3.1.1 详细模块介绍
项目整体分为三大模块：核心解析引擎（src/core/）、比较分析流水线（src/comparison/）和Web服务层（web/）。核心解析引擎提供Rust和C语言的AST解析能力，基于Tree-sitter提取函数级结构信息，计算AST结构指纹，并通过相似度模块计算Jaccard相似度和双向覆盖率。比较分析模块是系统的核心，实现了从AST特征提取、模块分类、相似度计算到多智能体博弈仲裁的完整流水线，包含文档分析Agent、演进史审查Agent、描述性文档Agent、全局真实性仲裁Agent、AST代码对标Agent等多个智能体角色。Web服务层基于Flask框架，提供任务提交、进度跟踪、报告展示、历史归档、源码溯源等完整功能，前端采用原生JavaScript实现单页应用体验。
