#### 3.1.1 详细模块介绍
项目围绕"在 StarryOS 上部署 ACT 模仿学习策略"这一核心目标，构建了从模型训练、ONNX 导出、Rust 推理器到 QEMU 集成的完整工具链。ACT 模型模块实现了完整的 CVAE + Transformer 架构，支持多相机输入和 Temporal Ensembling；Rust 推理器基于 tract_onnx 实现，支持多种运行模式和内存分析；minrt 原型提供了 Python 端的 ONNX 运行时参考实现；ONNX 分析工具链可对计算图进行 FLOPs、内存、融合等多维度分析；StarryOS QEMU 集成已验证端到端推理成功。
