#### 3.1.1 详细模块介绍
AgentJS 是一个基于 Rust 的轻量级 JavaScript 执行引擎，面向智能体场景中的短时、高频、即时执行任务。项目采用模块化架构，核心执行链为：source -> lexer -> parser / AST -> bytecode -> VM -> runtime / builtins -> JsValue。

**lexer 模块**：实现 ECMAScript 词法分析，支持 Unicode、正则表达式字面量、模板字符串等复杂词法规则，代码量约 93KB。

**parser 模块**：实现递归下降语法分析器，支持表达式（151KB）和语句（117KB）解析，包含完整的错误恢复机制。

**bytecode 模块**：包含编译器（242KB）将 AST 编译为紧凑字节码，定义 200+ 操作码，支持闭包、作用域、控制流等高级特性。

**vm 模块**：核心解释器（298KB）执行字节码，实现完整的指令调度、栈管理、函数调用、异常处理等。

**runtime 模块**：提供运行时上下文（161KB）、对象模型（41KB）、垃圾回收（5KB）、堆管理（14KB）、值类型（13KB）等基础设施。

**builtins 模块**：实现 ECMAScript 标准内置对象，包括 Array（63KB）、String（18KB）、Date（254KB）、JSON（26KB）、Promise（37KB）、Proxy（43KB）、Map/Set（114KB）、TypedArray（110KB）等，覆盖全面。

项目特色包括：轻量化 Native 构建（7.10 MiB）、分段稠密数组存储、属性描述符旁路表、Free List 堆对象复用、String Primitive ASCII Fast Path 等优化。
