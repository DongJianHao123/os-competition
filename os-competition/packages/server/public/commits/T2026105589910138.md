#### 3.1.1 详细模块介绍
项目accel-stack是一个透明的用户态+内核BPF协同TCP/IP协议栈加速层。它通过LD_PRELOAD机制拦截libc socket调用，在不修改应用程序源码或二进制的前提下，将UDP流量透明地切换到AF_XDP零拷贝数据路径，将同主机TCP流量通过BPF sockmap进行内核态重定向。项目分为7个里程碑（M1-M7），从基础的AF_XDP零拷贝接收开始，逐步构建了完整的透明加速层，包括UDP接收/发送加速、TCP sockmap代理、延迟优化（busy-poll、IRQ/CPU亲和性）、自研ARQ可靠传输协议和拥塞控制算法。每个模块都有详细的文档、benchmark数据和对应的git提交记录。
