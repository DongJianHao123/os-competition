#### 3.1.1 详细模块介绍
项目 yiliu_pathfinder 是一个面向 Agent 内存场景的磁盘向量检索系统，包含 Python 实验框架和 C++ 原生引擎两部分。Python 框架（src/yiliu_pathfinder/）提供完整的实验流水线：通过 pipeline.py 协调数据集加载、图构建（精确 kNN 或 LSH 近似）、页面布局生成（5 种策略）、页面存储（模拟或磁盘）、基准测试（顺序/并发/混合读写/增量插入）和结果输出。search.py 实现 best-first 图搜索算法，支持自适应预算策略。C++ 引擎（engine/）提供高性能的磁盘页面存储（page_store.cpp，支持 iostream 和 posix_pread 双后端）和图搜索（search.cpp），通过 CMake 构建为静态库和两个基准测试可执行文件。项目还包含完整的实验配置系统（experiment_config.py）、结果汇总（experiment_summary.py）、查询诊断（query_diagnostics.py）和 TEXMEX 数据集管理（texmex_corpus.py）。
