#### 3.1.1 详细模块介绍
项目包含9个递进式实验模块，覆盖OpenEuler操作系统核心子系统。实验一为基础环境搭建；实验二至实验九分别涵盖内核模块编程、进程管理、调度算法、内存管理、设备驱动、文件系统、网络通信、容器与智能调优。实验二、四、六融入了电力行业场景（智能变电站监控、电力调度、传感器驱动），体现了行业应用特色。代码组织采用doc/src双目录结构，部分实验（二、七、八）提供starter/solution双分支，便于教学。团队分工明确：金奕如负责实验一、四、六；宋雅笛负责实验二、七、八；邵奕负责实验三、五、九。
