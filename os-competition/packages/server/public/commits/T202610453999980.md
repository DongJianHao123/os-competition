#### 3.1.1 详细模块介绍
本项目围绕"启智01开发者套件（Ascend 310B NPU）"完成了从系统移植到AI推理的全链路开发。首先将OpenHarmony 5.0.3 Release移植到开发板，涉及源码获取、硬件适配包集成、hboot固件修改、HDMI/USB驱动适配和图形桌面启动。随后移植Ascend CANN 8.2.RC1推理引擎，包括NNRT运行时、Toolkit工具链、Python环境（numpy、opencv-python、ais_bench）的安装与配置。在应用层，团队开发了三种YOLOv5推理方式：Python CLI工具（支持图片检测、结果可视化）、Python RESTful API服务（支持HTTP远程调用）、C语言原生推理程序（基于ACL C接口，性能更优）。此外还提供了模型转换工具（ONNX→OM）、交叉编译脚本、一键部署运行脚本等配套工具。系统性能优化方面，通过内核裁剪、服务并行化、图形优化等手段，将启动时间从50s优化至44s，内存占用从1700MB降至1510MB，CPU使用率从8%降至5%。NPU推理性能达到单帧18ms、55.6 FPS、mAP@0.5达95.2%。
