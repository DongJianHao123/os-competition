#### 3.1.1 详细模块介绍
**Base DiskANN 外存图索引**：项目核心基于 DiskANN 架构，实现了 SSD 常驻图索引。`PipeANN/src/index.cpp` 包含完整的图构建（build）、近似最近邻搜索（search）、增量合并（consolidate）和 NGFix 精化（ngfix_refine）功能。代码使用 OpenMP 并行化、锁表并发控制，支持 float/int8/uint8 多种数据类型。图构建采用 Vamana 算法，搜索使用固定点迭代。

**读路径流水线执行层**：在 `PipeANN/src/search/` 目录下实现了多种搜索策略。`pipe_search.cpp` 实现了流水线搜索的核心逻辑，通过 `do_pipe_search` 方法协调异步 I/O 和距离计算。`beam_search.cpp` 实现波束搜索，`coro_search.cpp` 实现协程搜索，`page_search.cpp` 实现页级搜索。`pipe_search_common.h` 包含共享的流水线搜索基础设施。

**动态写入覆盖层**：在 `PipeANN/src/update/` 目录下，`direct_insert.cpp` 处理直接插入，`delete_merge.cpp` 处理删除和合并。写入覆盖层通过 WAL（Write-Ahead Log）保证持久性，内存 Delta 结构维护增量数据，tombstone 标记删除。查询时先搜索 Base 索引，再扫描 live Delta，过滤 tombstone 后合并 Top-K 结果。

**离线主动调参器**：通过 Python 脚本 `scripts/model_tune_pipegor.py` 和 `scripts/offline_tune_pipegor.py` 实现。读取历史 benchmark 记录，构建代理模型，使用贝叶斯优化选择最优流水线参数组合。

**Python 客户端库**：`PipeANN/pipeann/` 目录提供完整的 Python 接口，包括 `client.py`（客户端连接管理）、`collection.py`（集合操作）、`filter.py`（属性过滤）、`index.py`（索引管理）、`langchain.py`（LangChain 集成）、`qdrant_server.py`（Qdrant 兼容服务）。

**基准测试与实验脚本**：`scripts/tests-odinann/` 和 `scripts/tests-pipeann/` 包含可复现的实验脚本，覆盖图6-图18的论文实验数据，包含 hello_world 快速入门示例和 plotting 可视化工具。
