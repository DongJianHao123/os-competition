#### 3.1.1 详细模块介绍
项目借鉴多流固态硬盘（Multi-stream SSD）的设计思想，将KV缓存按访问热度划分为热、温、冷三个数据流，分别对应GPU显存、CPU内存、SSD外存三级介质。系统共设计4个核心模块：KV分页与页表管理模块负责缓存分页与映射；热度统计与访问监控模块负责量化每个缓存块的热度；多流动态迁移调度模块负责三级存储间的批量升降级；分池内存分配器模块负责按流划分独立内存池以减少碎片。目前项目处于阶段1完成状态，已完成环境搭建、基线推理跑通和方案设计定稿，尚未进入编码实现阶段。
